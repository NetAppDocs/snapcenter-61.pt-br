= Definir uma estratégia de restauração e recuperação para recursos do PostgreSQL
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você deve definir uma estratégia antes de restaurar e recuperar seu cluster para que possa executar operações de restauração e recuperação com sucesso.


NOTE: Somente a recuperação manual do cluster é suportada.

.Passos
. Determinar as estratégias de restauração suportadas para recursos PostgreSQL adicionados manualmente
. Determinar as estratégias de restauração suportadas para clusters PostgreSQL descobertos automaticamente
. Decida o tipo de operações de recuperação que você deseja executar.
