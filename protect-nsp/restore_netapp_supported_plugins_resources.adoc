= Restaurar recursos de plug-in suportados pelo NetApp
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
O fluxo de trabalho de restauração e recuperação inclui planejamento, execução de operações de restauração e monitoramento das operações.

.Sobre esta tarefa
O fluxo de trabalho a seguir mostra a sequência na qual você deve executar a operação de restauração:

image::../media/all_plug_ins_restore_workflow.gif[Fluxo de trabalho de restauração]

Você também pode usar cmdlets do PowerShell manualmente ou em scripts para executar operações de backup, restauração e clonagem.  Para obter informações sobre cmdlets do PowerShell, use a ajuda do cmdlet SnapCenter ou consulte o https://docs.netapp.com/us-en/snapcenter-cmdlets/index.html["Guia de referência do cmdlet do software SnapCenter"] .
