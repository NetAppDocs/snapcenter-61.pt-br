= Configurar a conta de armazenamento do Azure
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você deve configurar a conta de armazenamento do Azure no SnapCenter.

A conta de armazenamento do Azure contém detalhes sobre o ID da assinatura, a credencial do Azure e a conta do Azure NetApp .


NOTE: As licenças padrão e baseadas em capacidade não são necessárias para o Azure NetApp Files.

.Passos
. No painel de navegação esquerdo, clique em *Sistemas de armazenamento*.
. Na página Sistemas de Armazenamento, selecione * Azure NetApp Files* e clique em *Novo*.
. Selecione a credencial, o ID da assinatura e a conta NetApp nas respectivas listas suspensas.
. Clique em *Enviar*.
